Required HTML Elements

To ensure the page functions correctly, the following components must be included in the HTML:

  1. An input element with the ID falcon-address-input.
  2. A button element with the ID falcon-address-button.
  3. A custom tag: <falcon-scripts></falcon-scripts>.

Below are examples of these components:

Example 1: Input
<input id="falcon-address-input" type="search" class="form-control" value="">
Example 2: Button
<button id="falcon-address-button" type="button" class="btn w-100 btn-primary shadow lift">
    Get Offers
</button>
Example 3: Custom Tag
<falcon-scripts></falcon-scripts>
Dynamic Media Tokens

You can use the following tokens in your HTML to dynamically insert media that you've selected for this landing page:

Token Description
[[VideoUrl]] URL of your selected video
[[PrimaryImageWideUrl]] URL of your primary image in wide format
[[PrimaryImageTallUrl]] URL of your primary image in tall format
[[SecondaryImageWideUrl]] URL of your secondary image in wide format
[[SecondaryImageTallUrl]] URL of your secondary image in tall format

Below are examples of how to use these tokens in your HTML:

Example: Video Element
<video controls width="100%" class="shadow">
    <source src="[[VideoUrl]]" type="video/mp4">
    Your browser does not support the video tag.
</video>
Example: Responsive Primary Image
<picture>
    <source media="(min-width: 768px)" srcset="[[PrimaryImageWideUrl]]">
    <img src="[[PrimaryImageTallUrl]]" alt="Primary Image" class="img-fluid">
</picture>
Example: Secondary Image Gallery
<div class="row">
    <div class="col-md-8">
        <img src="[[SecondaryImageWideUrl]]" alt="Secondary Image Wide" class="img-fluid">
    </div>
    <div class="col-md-4">
        <img src="[[SecondaryImageTallUrl]]" alt="Secondary Image Tall" class="img-fluid">
    </div>
</div>
Note: These tokens will only work if you have selected media in the page editor. If no media is selected for a particular slot, the token will be replaced with an empty string.
Account Field Tokens

You can use the following tokens in your HTML to dynamically insert account information:

Token Description Example Value
[[AccountName]] Account/Company Name ABC Company
[[AccountAddress]] Street Address 123 Test Lane
[[AccountCity]] City Dallas
[[AccountState]] State (2 letter code) TX
[[AccountZip]] Zip/Postal Code 30872
[[AccountLicenseNumber]] License Number 112334-ACT
[[AccountEmail]] Contact Email [email protected]
[[AccountPhone]] Contact Phone (770) 345-1212

Below are examples of how to use these tokens in your HTML:

Example: Company Footer
<footer class="py-4 bg-light">
    <div class="container">
        <div class="row">
            <div class="col-md-6">
                <h5>[[AccountName]]</h5>
                <p>
                    [[AccountAddress]]<br>
                    [[AccountCity]], [[AccountState]] [[AccountZip]]
                </p>
                <p>License: [[AccountLicenseNumber]]</p>
            </div>
            <div class="col-md-6 text-md-end">
                <p>Contact Us:</p>
                <p>
                    <a href="mailto:[[AccountEmail]]">[[AccountEmail]]</a><br>
                    <a href="tel:[[AccountPhone]]">[[AccountPhone]]</a>
                </p>
            </div>
        </div>
    </div>
</footer>
Example: Contact Card
<div class="card shadow">
    <div class="card-header bg-primary text-white">
        Contact [[AccountName]]
    </div>
    <div class="card-body">
        <div class="d-flex align-items-center mb-3">
            <i class="fas fa-phone-alt me-2"></i>
            <span>[[AccountPhone]]</span>
        </div>
        <div class="d-flex align-items-center">
            <i class="fas fa-envelope me-2"></i>
            <span>[[AccountEmail]]</span>
        </div>
    </div>
</div>
Note: These tokens will be replaced with actual account information when the landing page is generated. They can be edited by Account Admins in the Account Info screen.
